Version Management Simplified

Automate and secure your application versioning with our robust API.

Features

  • Secure Version Control: Manage your applications securely with JWT-based access controls, ensuring that only authorized users can alter versions.
  • Dynamic Versioning: Easily bump, set, and track application versions with a simple API call.
  • Durable Storage: All version data is securely stored and managed through DynamoDB for reliability and speed.

API Documentation Preview

Creating an Application (Non-Secure and Secure Examples): 

curl -X POST https://yourapi.com/create -d 'app_name=example&secure=false'
curl -X POST https://yourapi.com/create -d 'app_name=secure_example&secure=true'

Retrieving the Current Version (Example for a Non-Secure App): 

curl https://yourapi.com/example/version

Bumping the Version (Secure and Non-Secure Examples): 

curl -X POST "https://yourapi.com/example/bump?type=minor"
curl -X POST "https://yourapi.com/secure_example/bump?type=major" -H "Authorization: YOUR_JWT_TOKEN"

Explanation: Non-secure apps require no authentication token. Secure apps, however, require a JWT token to be passed in the header for authorization. This ensures enhanced security by verifying that the requester has valid access rights to perform version management tasks on secure applications.
